- - -
+ ### 다운로드
  ##### 1. <span style="color:green">Code</span>
  ##### 2. Download ZIP
- - -
+ ### 실행방법
  ##### 1. Server 폴더 -> `Project.exe` 실행           (서버 구동)
  ##### 2. Client 폴더에 NetworkAddress.txt 파일 생성  (네트워크 확인)
  ##### 3. 첫번째 줄에 `IP`, 두번째 줄에 `Port` 입력   
  ##### 4. `2PM.exe` 실행                              (클라이언트 구동)
  ##### 5. Let's Play ~!                               (게임 즐기기)
